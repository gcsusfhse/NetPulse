# Screenshots & Sample Outputs

This folder contains sample screenshots and terminal output captures from NetPulse.

## Contents

| File | Description |
|------|-------------|
| `banner.png` | NetPulse startup banner in terminal |
| `scan_running.png` | Live scan progress output |
| `scan_results_table.png` | Discovered devices table |
| `csv_output.png` | Sample CSV file opened in spreadsheet |
| `report_output.png` | Generated text report |
| `port_scan.png` | Port scan results for a single host |

## Note for Contributors

When adding new screenshots:
- Use a terminal with dark background for consistency
- Capture at 1280x720 resolution minimum
- Use `scrot`, `gnome-screenshot`, or macOS's `Cmd+Shift+4`
- Name files descriptively: `feature_description.png`

> **Privacy**: Do not commit screenshots with real MAC addresses, hostnames,
> or IP addresses that could identify real devices on a personal network.
> Use the sample data from `sample_scans/` for demonstration screenshots.
